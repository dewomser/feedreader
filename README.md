## Usage
* copy both shellscripts in the same dir
* only get-rss1.sh needs to be edited.
* 1.line :path
* last line: browser
## Get Rss an example :
./get-rss1.sh worms https://www.wormser-zeitung.de/rss/lokales/worms/nachrichten-worms
